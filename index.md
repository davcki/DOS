# Prévision de la consommation d’électricité en France

Projet de data science réalisé durant ma licence.

## Objectif
Développer un outil Python permettant :
- de prédire la consommation électrique française à partir de séries temporelles
- de visualiser la consommation par département sur une carte

## Méthodes
- Analyse de séries temporelles
- Traitement de données énergétiques
- Visualisation cartographique

## Technologies
Python, Pandas, Matplotlib
